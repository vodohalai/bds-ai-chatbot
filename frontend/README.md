# Frontend (React UI)

## Mục đích
Giao diện chat hiện đại, thân thiện, tối ưu trải nghiệm người dùng, giống ChatGPT.

## Cấu trúc
- `src/`: Mã nguồn React
- `public/`: index.html

## Khởi tạo & phát triển
1. `npm install`
2. `npm start` (mở http://localhost:3000)

## Build production
- `npm run build`

## Triển khai trên Cloudflare Pages
- Build ra thư mục `dist/` và kết nối với Cloudflare Pages.
